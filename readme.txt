==================================================
  [310CardGame] ランチャー説明書
==================================================

■ 本ファイルについて
これは、アプリ本体の起動と、最新版の自動アップデートを行うためのランチャーです。
ご利用ありがとうございます！

■ 動作環境
- Windows 10/11

■ 使い方

1. Launcher.exeを**任意のフォルダ**に置いてください。
   ※ Cドライブ直下やデスクトップなど、書き込み権限のある場所に置いてください。

2. **Launcher.exe**をダブルクリックして実行します。

3. [初回起動時について]
   初回起動時は、アプリ本体（main.exeとデータ）が存在しないため、自動的にGitHubから最新版をダウンロードし、**「main」フォルダ**を作成します。
   この際、一時的に黒い画面が表示されますが、そのままお待ちください。

4. [アップデートについて]
   2回目以降の起動時、Launcher.exeが自動的にインターネット経由で最新バージョンをチェックします。
   最新版がある場合は、自動的にダウンロード・展開・上書きを行い、その後アプリ本体を起動します。

■ 注意事項
- アプリ本体（main.exeなど）は、Launcher.exeと同じフォルダ内に生成された「main」フォルダの中に格納されます。
- アプリ本体（main.exeなど）を直接起動せず、**必ずLauncher.exeから起動**してください。
- Launcher.exeを削除しても、mainフォルダ内のデータは残ります。
